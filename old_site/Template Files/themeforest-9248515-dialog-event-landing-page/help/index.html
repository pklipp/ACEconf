About

This theme is based on Twitter Bootstrap 3 grid system, so there are some additional files included.

index.html is the core file of the template.

  • php/process.php - this file makes the registration and subscribe forms work
  • php/subscribers.txt - here will apear a list of all people that subscribed via subscribe form
  • /css/ folder - In this folder you can find all .css files of this theme. If you would like to change some styles edit only style.css
  • /js/ folder - Here you can find all .js files. If you want to change some javascript files, you can only change main.js
  • /images/ folder - Here are located all images used in this theme
  • /media/ folder - This folder is only for video background, you can find there all video files.

 

If you have any question, problems with customisation feel free to send me a message on jenia@stratila.com

HTML Structure

In this template you can move different sections ( Section example: Speakers, F.A.Q, Schedule ) up and down, in the way it fits your needs.

A section look's like this:

<!--===============================-->
<!--== Section Title ==============-->
<!--===============================-->
Section Content
<!--==========-->

and it can be moved anywhere inside of this container:

<div class="container content">
    Sections Here
</div>

Do not place a section inside another section!

Color Scheme

If you want to change color scheme of this template, all you have to do is open css/style.css and edit this lines:

/*--------------------
Color Scheme =======*/

.preloader div, .preloader, .preloader:before, .preloader:after,
.button, a.register, .speaker-contacts, .highlight .table-header,
.owl-buttons, ul.social-icons li a:hover, .button-active:after {
    background-color: #7ab55c; /* <- Paste here your prefered color */
}

blockquote, input:focus, .page-title .box, .navbar-custom .navbar-nav > li > a:hover,
.navbar-custom .navbar-nav > li > a:focus, .navbar-custom .navbar-nav > .active > a,
.navbar-custom .navbar-nav > .active > a:hover, .navbar-custom .navbar-nav > .active > a:focus {
    border-color: #7ab55c; /* <- Paste here your prefered color */
}

.highlight .table-header:after {
    border-top-color: #7ab55c; /* <- Paste here your prefered color */
}

a {
    color: #7ab55c; /* <- Paste here your prefered color */
}

Replace "#7ab55c" with your prefered color.

Event Info

To edit event info, all you need to do is to replace default text strings in /js/main.js

eventInfo = ["Brussels, Belgium", "18 December 2014"]; // Event Info

you can add, as many text strings as you wish, just separate them.

Example: [ "First", "Second", "Third" ]

Photo Background

To change the images on the background you should find this code:

<!--== Header Background ==-->
<div id="header-background" class="header-background">
    <ul class="slides">
        <li><img src="images/demo/bg-slide-01.jpg" alt="Slide 1"></li>
        ...
    </ul>
</div>

here is how a single slide look's:

<li><img src="images/demo/bg-slide-01.jpg" alt="Slide 1"></li>

Just replace image path ( src="images/demo/bg-slide-01.jpg" ) with your image path.

You can add as many slides as you wish.

Video Background

Changing video background is as simple as replacing several files in media folder.

In order to html5 video work on each browser you need to have the video in 3 different formats: mp4, ogv and webm. You can use video converter for this: http://video.online-convert.com/ , set width and height to 1280 x 720 for a better view.

You can choose an awesome short video http://www.beachfrontbroll.com/ , it is completely free to use.

Remember to add a .jpg image, that will be shown on mobile devices that does not support video.

Menu

The order of the links in the navigation bar MUST match the order of the sections as laid out on the page or unexpected behavior will occur.

Each menu link should have a HREF targeting coresponding section ID in the DOM.

Example bellow: ( href="#faq" targeting id="faq" )
<ul class="nav navbar-nav">
    <li><a href="#faq">FAQ</a></li>
    ...
</ul>
<!--===============================-->
<!--== FAQ ========================-->
<!--===============================-->
<section id="faq">
    ...
</section>
<!--==========-->

Image Slider

Adding and removing slides is simple.

A slider HTML looks like this:

<div id="slider" class="owl-carousel fit-right">
    <img src="images/demo/slide-01.jpg" alt="Slide 1"/>
    ...
</div>
src="images/demo/slide-01.jpg" - path to your image, alt="Slide 1" - slide title

Tabs

Working with tabs is pretty simple as well.

This is how tabs HTML look. Each tab should have a HREF targeting a coresponding tab pane ID in the DOM.

Example bellow: ( href="#tab1" targeting id="tab1" )
<div class="content-tabs">

    <!-- Tab Navigation -->
    <ul class="nav nav-tabs">
        <li class="active"><a href="#tab1" data-toggle="tab">First Tab Title</a></li>
        <li><a href="#tab2" data-toggle="tab">Second Tab Title</a></li>
        ...
    </ul>

    <!-- Tab Content -->
    <div class="tab-content">
        <div class="tab-pane active" id="tab1">
            First Tab Content
        </div>
        <div class="tab-pane" id="tab2">
            Second Tab Content
        </div>
        ...
    </div>

</div>

Accordion

Accordion's are that nice collapsing panels, used in Schedule and F.A.Q Sections.

This is how accordion HTML looks like:

<ul class="accordion-group">
    <li>
        <a class="accordion-heading">
            <h4 class="accordion-title">Why should i attend this event?</h4>
        </a>
        <div class="accordion-body">
            <p>
                Lorem ipsum dolor sit amet, consectetur adipisicing elit. Consectetur, <a href="#">omnis</a> veritatis laboriosam <b>incidunt</b> officiis maiores unde accusamus impedit.
            </p>
        </div>
    </li>
    ...
</ul>

each panel:

        <a class="accordion-heading">

        </a>

Register

Our registration form, send's you an mail each time a user is registered, so all you have to do is open process.php

and edit this line:

// Your Email Address
$youremail = "Your Email Adress Here";
Example: $youremail = "example@example.com";
